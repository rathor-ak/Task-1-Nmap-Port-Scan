Task 1 – Scan Your Local Network for Open Ports

Step 1: Install Nmap

Installed Nmap from official site.

Step 2: Find Local IP Range

Used ipconfig to get IP: 192.168.1.5 → IP Range: 192.168.1.0/24

Step 3: Run TCP SYN Scan

nmap -sS 192.168.1.0/24
192.168.1.1 → Ports open: 80, 443, 22, 53
192.168.1.5 → Ports open: 135, 445

Step 4: Analyze with Wireshark

Captured TCP handshakes, DNS requests, and HTTP traffic during scan.

Step 5: Services on Ports

  • 80 → HTTP
  • 443 → HTTPS
  • 22 → SSH
  • 53 → DNS
  • 135/445 → Windows SMB

Step 6: Security Risks Identified

  • Exposed SSH can be brute-forced
  • Unsecured HTTP might leak data
  • SMB ports are vulnerable to ransomware

Step 7: Save Results

Text: nmap -sS 192.168.1.0/24 -oN scan_results.txt

HTML: nmap -sS 192.168.1.0/24 -oX scan.xml && xsltproc scan.xml -o scan_results.html
